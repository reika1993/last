# 概要

保護ねこと里親をつなぐプログラムを作成します。

## 起動手順

catsテーブルのレコードは以下のとおりです。

```mysql
mysql> select * from cats;
+----+-----------+-----+--------+
| id | name      | age | sex    |
+----+-----------+-----+--------+
|  1 | おもち    |   2 | メス   |
|  2 | コア      |   3 | オス   |
|  3 | ゴンち    |   5 | オス   |
+----+-----------+-----+--------+
3 rows in set (0.00 sec)

```

ポストマンで起動確認をします。
